# Ethereum Bridge Contract

## Required

- **Docker** : Install docker [Official Docker](https://www.docker.com/).
- **Node.js** : Node v18.16.0 Use nvm node version manager.

## Install

`npm install`

## Build

`npm run compile`

## Deploy

`npm run deploy`

## Build wrappers

`make gen`

## Export wrappers to hyperion folder

Required ../hyperion repository

`make exportwrappers`
